# Plantilla para el Modo 'Mejorar'

### Descripción
Esta plantilla se utiliza para mejorar un prompt proporcionado por el usuario. La plantilla incluye instrucciones detalladas sobre cómo optimizar el prompt para obtener mejores resultados.

### Plantilla Base
```markdown
# Instrucciones para Mejorar un Prompt

Toma el prompt proporcionado por el usuario y mejóralo siguiendo estas pautas:

1. **Estructura del Prompt**:
   - Utiliza un lenguaje natural y descriptivo.
   - Evita el uso de etiquetas meta como "8K", "masterpiece", etc.
   - No utilices prompts negativos; describe solo lo que deseas ver.

2. **Componentes a Mejorar**:
   - **Claridad**: Asegúrate de que cada descripción sea clara y específica.
   - **Detalle**: Añade detalles adicionales que puedan mejorar la calidad del prompt.
   - **Consistencia**: Asegúrate de que todas las descripciones sean consistentes entre sí.
   - **Optimización**: Elimina cualquier descripción redundante o innecesaria.

3. **Ejemplo de Estructura**:
   ```
   Prompt original: "Una mujer con un vestido rojo."
   
   Prompt mejorado: "Un primer plano de una mujer adulta de etnia [etnia], con piel [tono de piel], en sus [edad]. Tiene el cabello [descripción del cabello] y ojos [color y forma de los ojos]. Lleva puesto un vestido rojo de [material], con un escote [tipo de escote] y mangas [tipo de mangas]. La imagen está tomada desde un ángulo [ángulo de la cámara], con una profundidad de campo [profundidad de campo]. El fondo es [descripción del fondo], con una iluminación [descripción de la iluminación]."
   ```

4. **Instrucciones Adicionales**:
   - Sé específico y detallado en cada descripción.
   - Utiliza términos técnicos de fotografía para describir la composición y la iluminación.
   - Asegúrate de que todas las descripciones sean visibles y concretas.
